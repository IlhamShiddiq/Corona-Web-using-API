# Corona Website

Show statistic data about Corona cases in Indonesia

## Installing

Install this project with npm Command Line

```
npm install
```

### Built With

* [Bootstrap](https://getbootstrap.com/) - The web framework used
* [Webpack](https://webpack.js.org/) - Module bundler
